* EitoKs/unity-lookat について *

このリポジトリはUnityで制作した作品のスクリプトをまとめた場所である。
なお、ここにあるすべてのソースプログラムは個人で作成した。
-----------------------------------------------------------------------
〇各フォルダについて
このリポジトリには計８個のフォルダが存在し、
以下に各フォルダの中に入っているファイルらの大まかな説明を示す。
（左に書かれた数字はリポジトリ内のフォルダを上から数えた数字である。）

①Animation
　キャラクターや敵のアニメーションを管理するソースプログラムが入っているフォルダ。

②Audio
　音関係のソースプログラムが入っているフォルダ

③Endroal
　エンドロールで使用したソースプログラムが入っているフォルダ

④Event
　イベント作成の際使用したソースプログラムが入っているフォルダ。

⑤Game
　メインゲームで使用したソースプログラムが入っているフォルダ。

⑥Title
　タイトル画面のオブジェクト管理で使用したソースプログラムが入っているフォルダ。


⑦huki
　文章を吹き出しで表示するためのソースプログラムが入っているフォルダ。

⑧other
　その他、ゲーム全体を管理するもの、ポーズ画面、フェードアウト処理などの
操作を行うために使用したソースプログラムが入っているフォルダ

-------------------------------------------------------------------------------
〇フォルダ内のファイルの説明について
　各フォルダ内にあるファイルの説明はファイル内のコメント文にて示す。
